Minhas habilidades

Front-End

JavaScript

A principal linguagem de progamação da web.

NextJs

Um dos principais frameworks para desenvolvimento full stack, nextJs facilita a o desenvolvimento Front-End e agiliza a comunicação entre Front-End e Back-End com seu sistema de Api's.

React

Biblioteca dinâmica e estável que flexibiliza e encurta o tempo de criação de interfaces.

Bootstrap

Uma das principais bibliotecas de estilização no mundo, simples e fácil de utilizar, auxilia eximiamente na criação de interfaces responsivas e agradáveis ao usuário.

Sass

Uma ferramenta facilitadora e simbólica, todavia auxilia na estilização de projetos e melhora a legibilidade do código.

Back-End Techs

Node

Um ambiente Javascript fácil de utilizar que permite a customização e escalabilidade para todo e qualquer tipo de projeto web.

Postgres

Um banco de dados relacional de fácil manejar e estrutura simples.Ótimo para projetos performáticos.

Express

Uma biblioteca do npm que auxilia na criação e organização do servidor Nodejs.

TypeScript

Um superset do Javascript que permite a tipagem dinâmica e estável, evitando o possível ocorrimento de bugs no sistema.

Prisma

Um ORM(Object Relational Mapping), ou mapeamento de objetos relacionais, é uma ferramenta que auxilia significativamente na manipulação e organização de dados com o Banco de Dados.
